# Fengshui And Direction Knowledge Card

source_id: `SRC-PROJECT-FENGSHUI-DIRECTION-SYNTHESIS`

这个模块用于报告里的方位、空间、城市、办公位置、居住习惯和轻量开运建议。它是辅助层，不是主判断层。

## 使用范围

- 可写：五行补偏、空间习惯、办公/学习位置、出行方向、城市气质、居住环境对节奏和情绪的影响。
- 可写：基于八字喜忌、紫微宫位主题、西占轴线和现实生活目标，给出低风险的环境调整建议。
- 可写：城市选择里的气候、行业、客户密度、交通、生活成本和资源网络，再辅以五行象意。
- 不写：没有现场勘测、罗盘坐向、户型图和长期居住反馈时，不做传统阳宅风水强断。
- 不写：不承诺“换方向必发财”“摆物必转运”“某方位一定有灾”。

## 证据顺序

1. 现实目标：读者要改善的是睡眠、专注、客户、合作、身体恢复、输出效率还是城市发展。
2. 结构化 facts：八字五行、喜忌倾向、大运流年、紫微宫位和西占轴线只作象意锚点。
3. 生活场景：办公桌、卧室、通勤、城市气候、客户来源和社交半径。
4. 低风险建议：先写可观察、可撤回、低成本的调整。
5. 校准反馈：建议执行 2-4 周后再看睡眠、效率、情绪、客户响应或合作摩擦是否变化。

## 写作边界

- 方位建议必须写成“更适合试”“可以优先考虑”“不建议长期压在某种环境里”，不能写成绝对命令。
- 城市判断先商业后命理；方位象意只能辅助解释，不能替代市场和现金流。
- 开运建议优先写行为和环境，不写昂贵物品、迷信摆件或恐吓式化解。
- 如果没有出生地、居住地、户型、坐向或执行反馈，只能写一般性空间偏好和校准问题。
- 高风险领域仍降级：医疗、法律、投资、婚恋重大决定不能用方位建议替代专业判断。

## 复盘采集点

- 调整后睡眠、专注、效率、沟通或客户响应是否有可描述变化。
- 哪条建议太玄、太泛或无法执行。
- 哪条建议其实来自现实生活结构，而不是方位象意。
- 城市/方位判断有没有被后续现实验证或推翻。
- 适用限制是什么：行业、预算、房型、通勤、家庭成员、健康状态或季节。
