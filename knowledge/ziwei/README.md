# Ziwei Rules

紫微知识卡只写星曜、宫位、四化、大限的解释边界。

默认边界：

- 命宫、身宫、宫位星曜、大限必须来自结构化 JSON。
- 单星不单断；至少结合宫位、三方四正、四化、大限和现实问题。
- 时间不准时，命身宫和大限宫位要写成倾向，并放进校准问题。

## 规则卡

- `foundations.md`: 单星不单断、命身/大限分层、四化是动点。
- `classical-anchors.md`: 《紫微斗数全书》与赋文提炼的方法锚点。
- `quanshu-vs-quanji-boundary.md`: 《紫微斗数全集》馆藏线索、不可晋升边界和与《全书》的使用区分。
- `palaces-stars-four-transformations.md`: 十二宫现代翻译、星曜边界和四化写法。
