# Ziwei Palaces, Stars And Four Transformations

source_id: `SRC-ZIWEI-PROJECT-SYNTHESIS`

## 十二宫现代翻译

| 宫位 | 现代问题 | 报告重点 |
| --- | --- | --- |
| 命宫 | 自我角色、第一反应、人生默认界面 | 写人格主轴，不写固定命运 |
| 兄弟 | 同辈、朋友、团队、竞争关系 | 写协作、比较和资源共享 |
| 夫妻 | 亲密关系、合作关系、长期互相要求 | 写沟通、边界、节奏 |
| 子女 | 创造物、作品、下属、照顾关系 | 写输出、作品和责任延伸 |
| 财帛 | 收入方式、金钱感、资源交换 | 写预算、回款、定价和风险边界 |
| 疾厄 | 身心压力、恢复方式、生活习惯 | 只写精力管理和就医意识 |
| 迁移 | 外部世界、出行、跨圈层机会 | 写外部场域和对外形象 |
| 交友 | 社群、人脉、客户、团队网络 | 写合作质量和圈层筛选 |
| 官禄 | 职业角色、职责、社会评价 | 写适合环境和验收方式 |
| 田宅 | 家庭、居住、资产感、安全基地 | 写稳定感和空间安排 |
| 福德 | 精神秩序、休息、内在满足 | 写内耗、恢复和长期价值感 |
| 父母 | 权威、制度、长辈、背书 | 写规则、资质和支持系统 |

## 星曜报告边界

- 主星写角色气质和策略，不写宿命结果。
- 辅星写资源、帮助、修饰能力。
- 煞星写压力、摩擦、速度和需要管理的风险，不写恐吓。
- 空劫、羊陀、火铃等只在多重证据支持时写成现实风险，并提供应对动作。

## 星曜置信度

- 单颗主星只给角色语言，不给事件结果。
- 主星、辅星、煞曜、四化在同一宫位或三方四正里共同指向时，才提高判断强度。
- 煞曜必须翻译成可管理的现实成本，例如速度、摩擦、冲动、延迟、压力、破旧立新，不写恐吓词。
- 桃花、贵人、文昌文曲等辅助象意不能直接写成“必有人帮”“必有桃花”；要落到社交方式、表达包装、资源入口或审美优势。
- 空劫类象意必须写成“预期落差、资源断点、重构成本”，不得直接写成失去、破败或灾。

## 四化现代翻译

| 四化 | 现代语言 | 写作边界 |
| --- | --- | --- |
| 化禄 | 资源、好感、流入、愿意投入 | 不等于一定发财 |
| 化权 | 权责、推动、控制、掌控感 | 不等于一定升职 |
| 化科 | 名声、可见度、文书、专业包装 | 不等于一定出名 |
| 化忌 | 卡点、牵挂、债务感、反复修复 | 不等于灾祸 |

## 四化写作步骤

1. 先写落在哪个宫位，也就是哪个生活领域被推到台前。
2. 再写是哪种动点：资源、权责、可见度或修复。
3. 再接现实场景：钱、关系、职业、家庭、学习、客户、社群或迁移。
4. 最后写行动建议：承接、分权、公开表达、补流程、还旧账或降低消耗。

禁用写法：

- 不写“化禄一定有钱”，改写为“这个领域更容易有资源流入或被人愿意投入，但能不能留下要看流程和能力承接”。
- 不写“化权一定升职”，改写为“权责变重，既可能是授权，也可能是被推到前面承担压力”。
- 不写“化科一定出名”，改写为“更适合做专业呈现、文书包装、公开说明和可信度建设”。
- 不写“化忌一定倒霉”，改写为“这里有反复牵挂、旧账或修复点，需要正面处理”。

## 与八字/西占合参

- 八字看到“能量结构”，紫微看到“领域舞台”。
- 西占看到“心理表达”，紫微看到“人生宫位”。
- 三者同向时可以写较强判断；三者不同向时写成内在拉扯和校准问题。

## 领域校准问题

- 命宫判断：这个角色感是别人对你的第一反应，还是你自己长期默认的处理方式？
- 官禄判断：职业压力更像职责变重、被看见、标准提高，还是方向不匹配？
- 财帛判断：钱的问题更像收入入口、定价方式、预算纪律，还是资源交换边界？
- 夫妻判断：关系问题更像亲密伴侣、长期合作方、客户关系，还是互相要求的模式？
- 迁移判断：外部机会更像城市、行业圈层、客户场域，还是对外形象变化？
- 福德判断：内耗更像精神秩序、休息方式、意义感，还是长期价值失焦？
