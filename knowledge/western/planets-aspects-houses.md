# Western Planets, Aspects And Houses

source_id: `SRC-WESTERN-PROJECT-SYNTHESIS`

## 行星功能

| 行星/点 | 功能 | 现实翻译 |
| --- | --- | --- |
| 太阳 | 核心意志、自我认同 | 你想成为什么、靠什么维持存在感 |
| 月亮 | 情绪、安全感、习惯反应 | 你累的时候需要什么、如何自我保护 |
| 水星 | 思考、表达、学习 | 你如何组织信息、沟通和判断 |
| 金星 | 喜好、关系、审美、价值感 | 你如何亲近、选择和感到值得 |
| 火星 | 行动、欲望、冲突处理 | 你如何争取、推进和发火 |
| 木星 | 扩张、信念、机会 | 你在哪些领域容易放大 |
| 土星 | 责任、限制、结构、长期训练 | 你在哪里需要成熟和建立标准 |
| 天王星 | 变化、突破、非典型 | 你哪里不愿被固定 |
| 海王星 | 想象、理想、模糊、共情 | 你哪里容易美化或失焦 |
| 冥王星 | 深层控制、危机、重塑 | 你哪里容易走极端或必须转化 |
| 上升 | 第一印象、进入世界的方式 | 别人一开始如何感受到你 |
| 天顶 | 职业形象、社会目标 | 你如何被外界验收 |

## 相位写法

- 合相：功能合并，主题强，但可能难分清。
- 六合/拱相：功能较顺，容易自然使用。
- 刑相：张力、摩擦、需要训练。
- 冲相：两端拉扯，常通过关系或外部场景显化。

报告中不要只写“有刑相所以不好”。正确写法是：哪个功能和哪个功能有张力，现实里会如何表现，怎么训练。

## 宫位写法

- 宫位只在出生时间和地点可靠时写。
- 宫位表示议题发生领域，不表示必然事件。
- 重要轴线：1/7 关系轴、4/10 家庭事业轴、2/8 资源共享轴、3/9 学习传播轴、5/11 创作社群轴、6/12 工作恢复轴。

## 写作边界

- 行星、相位和宫位只用于表达层、心理语言层和议题领域，不作为科学人格测评。
- 出生时间或地点不稳时，上升、天顶、宫位和宫主星只能写成待校准，不得写死。
- 涉及健康、投资、法律、婚姻等高风险议题时，只能写现实风险意识和沟通动作，不得给诊断、买卖点或重大决定。

## 与报告专题对应

- 事业：太阳、土星、天顶、十宫、六宫、二宫。
- 财富：二宫、八宫、金星、木星、土星。
- 爱情：金星、火星、月亮、七宫、五宫。
- 健康/精力：月亮、火星、六宫、十二宫；只写习惯和恢复。
- 人际合作：七宫、十一宫、水星、金星、土星。
- 学习成长：水星、九宫、三宫、土星、木星。
