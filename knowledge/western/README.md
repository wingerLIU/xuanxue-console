# Western Astrology Rules

西占知识卡用于心理语言、关系表达和职业形象，不替代八字/紫微事实层。

默认边界：

- 行星落座、相位、上升、天顶必须来自脚本 JSON。
- 未提供经纬度时，不写上升、宫位和天顶。
- 上升和宫位对出生分钟敏感，必须配合时间敏感性检查。
- 健康、投资、婚姻等高风险议题只给现实建议，不做决定替代。

## 规则卡

- `foundations.md`: 西占在本项目中的角色、时间敏感项和心理语言边界。
- `modern-psychological.md`: 现代心理占星语言、科学边界和报告禁区。
- `planets-aspects-houses.md`: 行星功能、相位写法、宫位轴线和专题对应。
