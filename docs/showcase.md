# Showcase Boundary

主仓库不放真实 case，也不放报告截图。

如果以后要做公开展示，建议单独准备一个 `showcase/export` 目录或独立仓库，并人工检查下面这些内容：

- 姓名、生日、城市、关系身份、聊天截图和本机路径都已遮掉。
- PDF / HTML 截图只展示版式、章节密度和阅读体验，不展示可反推当事人的原文。
- 示例文案使用虚构或充分改写后的资料。
- 展示图只说明能力边界：结构化排盘、事实复查、丰富版、简洁版、手机阅读、追问复盘。
- 不把展示页写成“保准”“代替决策”或强商业转化页面。

适合展示的方向：

- 一张报告封面。
- 一张目录或章节结构图。
- 一张 PDF 小图，说明丰富版和简洁版的字数层级。
- 一张手机阅读版截图，展示暖纸、深字、少色、无卡片的阅读风格。
- 一张流程图，说明真实资料只进入外部 run，不进入公开仓库。
